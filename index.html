Climate Classification Tool

Interactive Köppen-Geiger climate analysis using ERA5 data

Research-grade climate classification tool for analyzing global climate patterns. Features interactive mapping, time series analysis, and Köppen-Geiger classification.

Research Capabilities

  • Interactive Earth Engine climate mapping
  • Point and regional climate analysis
  • ECMWF ERA5 reanalysis data (1979-2022)
  • Automatic Köppen-Geiger classification
  • Temperature and precipitation time series
  • Climate trend analysis and data export
